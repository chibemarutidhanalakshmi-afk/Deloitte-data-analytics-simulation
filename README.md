# Deloitte Data Analytics Simulation

Completed Deloitte Data Analytics virtual simulation on Forage.

## Problem

Analyse machine data to identify unhealthy machines and areas with high failure rates.

## What I Did

* Imported JSON data into Tableau
* Created a calculated field to identify unhealthy machines
* Built visualisations to analyse machine failures
* Designed a simple dashboard

## Formula Used

* Created a calculated field:
  IF status = "unhealthy" THEN 1 ELSE 0 END

## Visualisations

* Unhealthy Machines by Factory
* Unhealthy Machines by Device Type

## Insights

* Some factories had higher machine failures
* Certain device types showed more issues

## Tools Used

* Tableau
* Microsoft Excel
